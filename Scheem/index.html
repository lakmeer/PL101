Welcome to my Scheem Interpreter

Scheem is a toy Scheme-like language developed as part of Nathan's University PL101 Programming Languages course.


Some instructions:

  • This interpreter is live-updating. Type in left side to change the program.
  • λ is supported as an abbreviation of lambda (use CTRL+L to type it)
  • Intesive recursive calculations will totally lock up your browser.
  • def is supported as an alias of define
  • program is an alias of begin. There is no compelling reason for this.
